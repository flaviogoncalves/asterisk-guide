# Legacy lab material (salvaged from the retired AsteriskTraining wiki)

These are the original hands-on labs from the 1st-edition course, preserved from
`github.com/flaviogoncalves/AsteriskTraining` (repo + wiki) before that project is
retired. They target **Asterisk 16** and use `chan_sip`/`sip.conf`, so they are
**reference material only** — to be modernized into the 2nd edition's per-chapter
"Lab" sections and the Asterisk 22 Docker lab in `../../lab/`.

- `wiki/` — the step-by-step lab guides (Lab 1–7, PJSIP lab, ODBC demo, FAQ).
- `configs/` — the matching `*.conf` files for each lab (LabNN/).

Mapping to the 2nd edition: Lab 1 → Installing; Lab 3.x → Building a PBX / Dialplan;
Lab 4 → PBX features; Lab 5 → SIP & PJSIP; Lab 6 → Dialplan advanced; Lab 7 → Security.
