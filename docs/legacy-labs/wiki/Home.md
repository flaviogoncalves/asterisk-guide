This lab requires experience with Linux, VirtualBox and networks. 

Required computer and operating systems:

Windows 10 with at least 2GB of free RAM and 20GBs of hard disk space
You will install softphones on Windows and Asterisk in the VirtualBox using Ubuntu 16.04

You may use a different setup, but at you own risk.

If the labs don't work as expected, check the FAQ below, try to troubleshoot using (core set verbose 15) and (sip set debug on) in the console. Pay close attention to not skip steps. 

[FAQ](https://github.com/flaviogoncalves/AsteriskTraining/wiki/FAQ) - Please check the FAQ, frequently asked questions before asking for tech support. 

Questions are usually answered in working days! 

I'm adding new labs for Asterisk 16, for now, just stick with Asterisk 15. Some labs using MACRO will not work on Asterisk 16. They will be changed some time in 2019.
