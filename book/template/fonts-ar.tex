% Arabic edition — RIGHT-TO-LEFT. pandoc puts the document in RTL mode from `dir: rtl` in the
% metadata (it loads babel + bidi for that). This header ONLY points the type at an
% Arabic-capable font via fontspec — which works whether the language package is babel or
% polyglossia. Amiri (from fonts-hosny-amiri) covers Arabic for body text and headings; code
% blocks, inline code, and URLs keep the Latin monospace font set in interior.tex.
%
% NOTE: do NOT use polyglossia commands here (\setotherlanguage, \newfontfamily\arabicfont …).
% pandoc loads babel for xelatex, so those are undefined control sequences and abort the PDF.
% Loaded AFTER interior.tex, so these override the TeX Gyre Latin faces for the Arabic build.
\setmainfont{Amiri}[Script=Arabic]
\setsansfont{Amiri}[Script=Arabic]
