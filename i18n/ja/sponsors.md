## 創設スポンサー — SipPulse {.unnumbered}

![](book/sponsors/sippulse-hero.png){width=100%}

**SipPulse は SIP の権威**であり、**200以上の通信事業者**を支える通信インフラです。
本書が Asterisk を教えるものであるのに対し、SipPulse は単一の PBX では不十分になったときに通信事業者が運用するものです。それは、**SoftSwitch、SBC、マルチテナント PBX、および音声AI IVR** を1つのスタックに統合したプラットフォームです。

*SipPulse を運用する事業者は、収益を最大化し、コストを削減し、安心して運用を行っています。*

### プラットフォーム {.unnumbered}

| 製品 | 機能 |
|---------|--------------|
| **SoftSwitch** | キャリアグレードの音声ルーティングおよびスイッチング — 最大 **1000 CAPS**。 |
| **SBC** | エッジを保護する Session Border Controller — 最大 **4000 同時通話**、Call Progress Analysis (CPA) を内蔵。 |
| **BCW** | マルチテナント PBX / 仮想 PABX およびコンタクトセンタープラットフォーム。 |
| **NIVA** | **ローカル**で動作する音声エージェントを備えた IVR — 自動化された AI 主導の音声応答。 |

### SipPulse Softphone {.unnumbered}

本書全体で使用されている **SipPulse Softphone** は、通信事業者向けのプロフェッショナルな **ホワイトラベル** ソフトフォンであり、**Android、iOS、Windows、macOS、Linux、および Web (WebRTC)** で音声、ビデオ、メッセージングを提供します。これは、**UDP、TCP、または TLS** を介して標準ベースのあらゆる Asterisk / PJSIP システムに登録でき、**SRTP** メディア暗号化と、バッテリー効率に優れた信頼性の高い配信を実現する **RFC 8599** プッシュ通知 (FCM / APNs) を備えています。対応コーデックには **Opus、G.722、G.711、iLBC、GSM、Speex** (オーディオ) および **H.264 / VP8** (ビデオ) が含まれます。

- データシート: <https://sippulse.com/datasheets/crosscall-v1.pdf>
- Web版を試す: <https://softphone.sippulse.com.br/crosscall/>
- 製品情報: <https://www.sippulse.com/produtos>

SipPulse は、本書の著者であり、SipPulse の CEO、そして **OpenSIPS Foundation** の創設メンバーである **Flavio E. Gonçalves** が率いています。詳細は <https://www.sippulse.com> をご覧ください。
